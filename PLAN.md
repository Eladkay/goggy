# Goggy
Goggy is a blogging platform built for simplicity and readability.
## Features
- Each Goggy server is built for a single blog administrator and many readers.
- Only the administrator can make posts. Everyone can read them.
- The editor is based on Markdown. The Markdown is rendered for the readers.
- Images are supported.
- The admin user can log in to create, delete and edit posts. 
- There's a menu bar and ways to view previous posts.

## Programming Notes
- The project's main language is Python and its main HTTP framework is FastAPI.
- Import any library needed, especially for HTTP handling and Markdown rendering
- Write CSS, JS and HTML as needed.
- Write the project in a maintainable way.
- Think of more things that I might need. Before implementing any extra features, ask me.
